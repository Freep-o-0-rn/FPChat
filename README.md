# FPChat MVP

Локальный MVP приватного текстового чата с инвайт-ссылками и клиентским шифрованием (E2EE-подход для MVP): сервер хранит только зашифрованные сообщения.

## Установка

```bash
npm install
cp .env.example .env
```

## Запуск

```bash
npm start
```

Приложение слушает только `127.0.0.1:3010`.

## Проверка порта 3010

```bash
ss -ltnp | rg 3010
```

## Открытие локально

- Основной экран: `http://127.0.0.1:3010/`
- Invite URL: `http://127.0.0.1:3010/i/<ROOM_PUBLIC_ID>#<SECRET>`

## Что хранит сервер

SQLite хранит:
- комнаты;
- участников;
- зашифрованные сообщения (`ciphertext`, `iv`);
- статусы `sent / delivered / read` и timestamps;
- verifier для recovery (salt + hash), без открытого recovery кода.

Сервер **не хранит в открытом виде** ключи комнаты/пользователя/recovery и текст сообщений.
